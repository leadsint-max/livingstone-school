# 🏫 Livingstone International School Management System

A full-stack, professional School Management System (SMS) built for scale and efficiency.

## 🚀 Deployment Guide
Follow the steps in `DEPLOYMENT_GUIDE.md` to launch this system on the cloud using Supabase, Render, and Vercel.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla API Integration)
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (via Supabase)
- **Security:** JWT (JSON Web Tokens) and Bcrypt Password Hashing
- **AI:** Python-based performance prediction engine

## 📂 Core Features
- **Administration:** Student admission with photo capture, staff management, and role-based permissions.
- **Academics:** Attendance marking, grade entry, and automated report card generation.
- **Finance:** Fee structure management and digital invoicing.
- **Operations:** Library, Hostel, and Transport tracking.
- **Communication:** Multi-user chat and notice board system.
- **AI Insights:** Predictive analysis of student failure risk.
